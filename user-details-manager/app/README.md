## Running the App

First, make sure to install all packages:

```bash
cd user-details-manager/app
npm install 
```

Then, you may run in dev mode (with hot reloading):

```bash
npm run dev
```

To run in built mode (faster, but without hot reloading):
```bash
npm run build
npm run start
```

To run the unit tests, run:
```bash
npm run test
```
